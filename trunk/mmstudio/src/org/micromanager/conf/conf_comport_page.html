Select the serial ports you use to control your devices. The port settings are shown at the right-hand side. Edit the settings using parameters specified in the documentation of the device. If you are unsure what the correct settings are, try: BaudRate=9600 baud, Parity=None, StopBits=1.

The AnswerTimeout parameter specifies (in milliseconds) how long Micro-Manager will wait for the response from the device. Values from 500ms to 2000ms are OK for most devices, but some (like the Nikon TE2000) can need up to 7000 ms.
