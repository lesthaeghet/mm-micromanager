All ScopeLED Illuminators

 

Property

VendorID

Type

Numeric value from 0 to 65535. The ScopeLED USB Vendor ID is 9410 (0x24C2).

Function

Sets the USB Vendor ID used to identify the target device. This property must be set before the device is initialized.

 

Property

ProductID

Type

Numeric value from 0 to 65535.

Function

Sets the USB Product ID used to identify the target device. This property must be set before the device is initialized.

 

Property

SerialNumber

Type

String value

Function

Prior to initialization, this property sets the Serial Number string used to identify the target device. It is optional to set this property.

After initialization, this property contains the Serial Number string of the device.

 

Property

LastError

Type

Numeric

Function

Read-only value indicating the last USB communication error. See Appendix A for a description of the error codes.

 

Property

LastDeviceResult

Type

Numeric

Function

Read-only value indicating the last device error. See your device documentation for a description of the error codes.

 

Property

TxnTime

Type

Numeric

Function

Read-only value indicating the number of milliseconds of the previous operation.

 

ScopeLED-MSB/MSP

 

Note: The ProductID for MSB/MSP illuminators is 4867 (0x1303).

 

Property

Channel1Brightness

Type

Floating-point value from 0 to 100, representing the percentage.

Function

Sets the brightness level of channel 1. This property changes the cached value. The new value will not take effect until the shutter is next opened.
